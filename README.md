# NEA
My A-Level NEA Project submitted in 2023 written in c++ using custom batch script for compiling.

* 2 player local co-op (keybinds assume using the same keyboard for both players), must stay reasonably close together or they lose
* randomised level generation, different types of terrain, e.g. flat, hills, gaps, magma (player takes damage)
* generated enemies run towards closest player, jumps if above
* wall behind the players forces them to attempt the level at a reasonable pace
* raylib v4.5 library for rendering and inputs

*Note: Used clangd format before uploading but was not used in submission.*
